Here is a basic example using the Arduino_GFX_Library and FT3168 library to get the Waveshare ESP32-S3 AMOLED 1.64 display running. Please copy files into their respective places in VSCode/PlatformIO and hopefully this will get you started.

The board and wiki are found here:

https://www.waveshare.com/ESP32-S3-Touch-AMOLED-1.64.htm

https://www.waveshare.com/wiki/ESP32-S3-Touch-AMOLED-1.64
